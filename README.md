## 🎯 Sobre o Projeto

**Insect Hunter Premium** é uma versão melhorada do clássico jogo "Catch The Insect". Transformamos um simples jogo de clicar em uma experiência completa com:

- ✨ **Efeitos visuais premium** (glitch, partículas, cards 3D)
- 🎮 **Sistema de combo** para multiplicar pontos
- ⚡ **Dificuldade dinâmica** (Fácil, Normal, Difícil)
- 🏆 **Ranking local** com as 10 melhores pontuações
- ⌨️ **Controles completos** (mouse, teclado, touch)
- 📱 **Design responsivo** para todos os dispositivos

## 🎮 Como Jogar

1. **Inicie o jogo** clicando em "Jogar"
2. **Escolha seu inseto** - cada um tem características únicas:
   - 🪰 **Mosca**: Rápida, poucos pontos
   - 🦟 **Mosquito**: Muito rápido, pontos médios
   - 🕷️ **Aranha**: Lenta, muitos pontos
   - 🪳 **Barata**: Veloz, muitos pontos
3. **Clique nos insetos** que aparecem na tela
4. **Mantenha o combo** clicando rapidamente
5. **Use ESC** para pausar e **R** para reiniciar
